# Photo Card UI Styling

## 개요

- 지금은 row당 desktop 환경에서 3개 mobile 뷰에서는 2개 이런식이다.
- 뭔가 사진을 올리고 감상하는 사이트인데 사진을 더욱 크게보고 클릭하고 싶게 만들어야할 것 같다.
- 무한스크롤을 지정해둬서 row당 2~3개로 나와버리면 사용성에 있어 약간 애매할 것 같다.

## 프롬포트

- 어차피 데스크톱과 모바일뷰 혼용가능한 레이아웃으로 배치되어 있기 때문에 row당 1개 씩 표시하는 가장 기본적인 SNS 처럼 처리해도 좋을 것 같다.
- 지금은 사진 카드형태에 버튼들이 오버레이 되어 카운튼 이런게 보이는 형태인데 카드 하단에 레이어를 하나 추가해서 그런 보기 옵션들은 그 레이어에 표기하자 (이미지 하단부에 뷰카운트, 좋아요 수, 댓글 수, 그리고 공유링크 만든거랑, 추후 다운로드 기능까지 넣도록하자)
- 그리고 포토카드 상단에 누가올렸는지 프로필 표시하고 우측 끝에 메뉴버튼을 하나 추가해서 드롭다운 기능하나 넣자 (해당 사진 안보기, 신고하기 등등 여타 사이트들 있는 그런 기능들로 확장되도록)
- 스타일은 알아서 현재 서비스 (보라테마)에 맞게 외부 사이트들의 카드폼 느낌 참고하여 이쁘게 만들어줘
